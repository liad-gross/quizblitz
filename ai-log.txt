Prompt: Write a Vue 3 single-file component called StartScreen. It should display a heading that says 'QuizBlitz', a short tagline, and a button labelled 'Play'. When the button is clicked, it emits an event called 'start'. Use the Options API (not Composition API). No props needed
Response: <template>
  <div>
    <h1>QuizBlitz</h1>
    <p>Test your web dev knowledge across 10 questions.</p>
    <button @click="$emit('start')">Play</button>
  </div>
</template>

<script>
export default {
  name: 'StartScreen',
  emits: ['start']
}
</script>
What I used: I used the code above because as the instructions stated it uses $emit and @click

Prompt: "Write a Vue 3 single-file component called ScoreBoard. It receives a prop called score (Number). It displays a heading 'Game Over', the player's score out of 10, and a 'Play Again' button. When Play Again is clicked, it emits 'restart'. Use the Options API."

Response: <template>
  <div>
    <h2>Game Over</h2>
    <p>You scored {{ score }} out of 10</p>
    <button @click="$emit('restart')">Play Again</button>
  </div>
</template>

<script>
export default {
  name: 'ScoreBoard',
  props: {
    score: Number
  },
  emits: ['restart']
}
</script>
